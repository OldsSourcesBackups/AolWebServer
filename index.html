About the Developer Help Link

The Developer Help Link is a collection of documents in the AOLserver Source Distributionintended for developers and system administrators who need information about the more intricate workings of AOLserver and its modules. The goal of the Developer Help Link is to provide detailed documentation of the code that is easy to find, easy to read, and easy to navigate. To that end, the files are editted in-place with the Mozilla Composer application and are directly linked from the "index.html" file at the top of the source distribution.
