About the Developer Help Link

The goal of the Developer Help Link is to provide detailed documentation of the code that is easy to find, easy to read, and easy to navigate. To that end, the files are editted in-place with the Mozilla Composer application and are directly linked from the "index.html" file at the top of the source distribution.
